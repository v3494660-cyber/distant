Программа считывает три числа из ввода и использовать модуль math для выполнения различных математических операций.

Формат ввода
Три числа, разделённые пробелами:

первое - вещественное число (например, 3.14159)
второе - целое число (например, 16)
третье - вещественное число (например, 5.8)
Работа самой программы:
Выведите три значения, каждое на отдельной строке:

Значение первого числа, округляется до 2 десятичных знаков
Из второго числа
Значение третьего числа, округлённое вниз до ближайшего целого

Для тестирования библиотек используется pytest

Пример работы программы:
23 45 43
23.0
6.708203932499369
43
